# Social Media Backend Engine

A scalable backend system built using Java 17 and Spring Boot that simulates social media interactions with concurrent request handling, Redis-based virality scoring, and PostgreSQL persistence.

## Features
- REST APIs
- Concurrent request handling
- Redis caching
- Virality scoring engine
- Dockerized environment
- PostgreSQL integration
- Thread-safe operations

## Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Redis
- Docker
- Maven
